"내 업무"는 모든 프로젝트에 걸쳐 나에게 할당되거나 이관된 업무를 한 곳에서 보는 페이지입니다.

현재 구현된 기능:

[내 업무] 
나에게 할당된(assignee_id = 나) 모든 카드를 프로젝트 구분 없이 한 목록으로 표시
필터: 전체 / 진행중 / 완료 / 기한초과
이관받은 업무는 "← 누구 이관 (처리결과)" + 코멘트 표시
마감일 기준 정렬 (기한초과 최상단)
사용 시나리오:

여러 프로젝트에 참여하는 사람이 "오늘 내가 해야 할 일"을 한눈에 파악
누군가 나에게 업무를 이관했을 때 바로 확인
기한 초과된 업무를 빠르게 식별
프로젝트 페이지는 "프로젝트 전체 현황"을 보는 것이고, 내 업무는 "나 개인의 할 일 목록"을 보는 것입니다.

[캘린더] 
마감일 기준으로 카드를 날짜별로 배치하여 보여주는 뷰입니다. 
현재는 마감일이 있는 카드를 날짜순으로 그룹화하여 리스트 형태로 표시합니다. 
"어떤 날에 어떤 업무가 마감인지" 한눈에 파악하는 용도입니다.

[대시보드]
마감 임박 기준은 오늘부터 3일 이내입니다.
대시보드의 "마감 임박" 목록: 마감일이 오늘~3일 후 사이인 미완료 카드
알림 스케줄러: D-3(3일 전), D-1(1일 전), D-day(당일) 알림 발송
예시 (오늘이 5/20이면):

마감일 5/20 → 마감 임박 (D-day)
마감일 5/21 → 마감 임박 (D-1)
마감일 5/23 → 마감 임박 (D-3)
마감일 5/24 → 마감 임박 아님
마감일 5/19 → 기한 초과

0
마감 임박(D-3 이내) 카드에서 워크플로우 이전 담당자가 아직 완료하지 않은 경우, 이전 담당자에게 자동으로 독촉 알림을 발송합니다.

동작 조건:

카드 마감일이 3일 이내
워크플로우가 설정되어 있음
현재 담당자가 워크플로우 2번째 이후 단계
이전 단계 담당자가 아직 완료 이관하지 않음
알림 내용: "업무 처리 요청 (마감 임박) - '{카드 제목}' 업무의 마감이 임박했습니다. 빠른 처리를 부탁드립니다."
